Use the 00 sequence for debugging purposes:

```
./
└── data/
    └── debug/
        ├──sequences
            ├── 00/           
            │   ├── velodyne/
            |   |	├── 000000.bin
            |   |	├── 000001.bin
            |   |	└── ...
            │   └── labels/
            |       ├── 000000.label
            |       ├── 000001.label
            |       └── ...
```
